# 🚀 FastAPI Integrated with ML Model

Production-ready Machine Learning API built using **FastAPI** and **Scikit-learn** for real-time predictions.  
Designed with clean architecture, request validation, and interactive API documentation to demonstrate practical ML deployment skills.

## ⚡ Tech Stack
Python • FastAPI • Scikit-learn • Pandas • NumPy 

## 🔥 Highlights
- Integrated trained ML model with FastAPI backend
- Built scalable REST APIs for prediction serving
- Implemented data validation using Pydantic
- Interactive Swagger API documentation
- Lightweight and deployment-ready architecture

---


## 🎯 Project Overview

This project demonstrates the development of a deployable Machine Learning API using FastAPI and Scikit-learn for intelligent insurance premium prediction.

The system simulates a real-world risk assessment workflow by transforming user data into predictive insights through scalable REST APIs and efficient backend processing.
